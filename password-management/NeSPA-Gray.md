# NeSPA-Gray

## Features

- One masterpassword for millions of passwords
- Simple version of NeSPA-Blue and NeSPA-Red
- Fix charset, policy, counter
- Identical on Windows, Linux and Android OS
- MVP / POC as desktop and mobile app
